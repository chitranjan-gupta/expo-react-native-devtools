# Expo React Native Devtools

## How to 

1. Install Extension from Chrome or Edge (Chromium Browser)

-- https://chromewebstore.google.com/detail/expo-react-native-devtools/hipipalomafpfjofeboocnchdnffjoeh
-- https://microsoftedge.microsoft.com/addons/detail/expo-react-native-devtool/aakbhgdpbgkgdijllkpoaolnejhobjkl

2. Run the Expo React Native Dev Server
```sh
npx expo start --tunnel --go
```

3. Copy the tunnel url and paste in extension

> It does not work for firefox
